Exercícios - Seção 1

  1. Crie uma barra de navegação com links estilizados:
    • Crie uma barra de navegação horizontal com pelo menos 5 links.
    • Adicione um link que abre em uma nova aba (target="_blank").
    • Use pseudo-classes (:hover, :active, :visited) para estilizar o comportamento dos links.
  2. Links internos e âncoras:
    • Crie uma página longa com seções diferentes.
    • No topo da página, crie um índice com links que naveguem diretamente para diferentes seções da página usando âncoras (#id).
    • Adicione um link que retorne ao topo da página em cada seção.
  3. Imagens responsivas com <picture> e srcset:
    • Adicione três versões de uma mesma imagem (pequena, média e grande) usando o elemento <picture> e o atributo srcset para exibir a versão correta com base no tamanho da tela.
  4. Imagens com legendas e figcaption:
    • Crie uma galeria de fotos com legendas usando o elemento <figure> e <figcaption>.
    • Adicione um efeito de zoom na imagem ao passar o mouse e mantenha a legenda sempre visível.

O que eu aprendi que achei interessante:

  • Existem elementos vazios, como a tag <br>. Elementos vazios não tem conteúdo e nem tag de fechamento;
  • O termo <!DOCTYPE html> informa que o documento web é no formato HTML5;
  • HTML não diferencia letras maiúsculas e minúsculas. A tag <p> é o mesmo que <P>;
  • Qualquer tag pode ter atributos;
  • Atributo "title": é um texto que é exibido ao deixar o mouse por cima do conteúdo do elemento (passe o mouse aqui em cima como exemplo);
  • Os navegadores adicionam uma margem top e buttom automaticamente para tags de título e parágrafo;
  • Alguns elementos de formatação interessantes:
    • <b>- Texto em negrito
    • <strong>- importante
    • <i>- Texto em itálico
    • <em>- Texto enfatizado
    • <mark>- Texto marcado
    • <small>- Texto menor
    • <del>- Texto deletado
    • <ins>- Texto inserido
    • <sub>- Texto subscrito
    • <sup>- Texto sobrescrito
  • A diferença entre é as tags <b> e <strong> é que <strong> tem "importância" (Semântica);
  • A diferença entre é as tags <i> e <em> é que <em> tem "importância" (Semântica);
  • Quando se trata de imagens, você pode utilizar a tag <picture> para quando precisar utilizar uma imagem diferente para cada tamanho de tela;

2. Importante sobre LINKS

  • Os links são definidos pela tag <a>;
  • O atributo target define como deve ser acessado o conteúdo do link. Oções de valores para o atributo:
    • _self- Padrão. Abre o documento na mesma janela/guia em que foi clicado
    • _blank- Abre o documento em uma nova janela ou aba
    • _parent- Abre o documento no quadro pai
    • _top- Abre o documento no corpo inteiro da janela
  • Você pode criar marcadores nos links. Por exemplo, quando você quer que um link direcione o usuário para uma parte específica da página web atual que tenha determinado id, basta colocar como valor para o href "#" e em seguida o id. Exemplo: href="#id1".
